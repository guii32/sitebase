Fábrica de Sonhos

O Que É?
--------------------------------------------------------------------------------------------------------------------

Fábrica dos Sonhos é uma eletiva da escola Professor Clemente Quáglio que procura desenvolver as habilidades tecnologicas dos estudantes com o intuito de contribuir para a carreira profissional ou rumo pessoal do aluno.

O Que Se Aprende?
  • Programação em HTML e CSS
  • Funcionalidade de sistemas elétricos solares
  • Impressão 3D
  • Impressão de estampas
  • Robótica simples
  • Mecatrônica

O Que Foi Feito?
  • Placa Solar Caseira
  • Sites em HTML e CSS (como este aqui, :D)
  • Carros movidos a controle remoto e Bluetooth
  • Maquinários Simples
  • Camisetas e canecas estampadas
